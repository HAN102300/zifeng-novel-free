# 紫枫小说 UI 重构 · 概览

## 完成内容
为 zifeng-web 全栈项目产出全新 UI 重构设计方案，方向为「品牌紫 + 玻璃拟态 + 复杂动画」。

## 交付物
- `design/zifeng-ui-redesign.html` — v1.0 交互式原型（导航栏 + 组件库 + 首页/书架原型）
- `design/REDESIGN-SPEC.md` — v1.0 设计规范文档
- `design/zifeng-pages-deep-dive.html` — v2.0 9 大页面布局深化方案

## v2.0 深化的 9 个页面
1. 首页六大榜单（左大图+右4列复合布局）
2. 榜单详情页（侧栏切换 + 排行榜列表 + 分页）
3. 分类页（男生蓝/女生粉双频道 + 3×3 分类网格）
4. 分类详情页（左侧筛选 + 右侧工具栏 + 4列网格）
5. 书架（阅读历史时间轴 + 收藏5列网格）
6. 阅读页（羊皮纸护眼 + 设置抽屉 + 进度条）
7. 书源列表（4统计卡 + 搜索/导入 + 列表项开关）
8. 设置页（6类导航 + 主题色/字号/玻璃效果控件）
9. 小说详情页（Hero + 双栏：简介章节/数据推荐）

## 关键设计决策
- 主色：紫枫紫 #8B5CF6，品红 #EC4899 高亮辅色
- 6大榜单视觉权重区分：主榜(必读/搜索/评论)用大图复合布局，小榜(潜力/更新/完结)用纯网格
- 阅读历史用横向时间轴节点，比传统列表更有「阅读轨迹」叙事感
- 阅读页保留 8 套背景预设（白/护眼绿/护眼黄/灰/暗夜/深黑/羊皮纸/淡蓝）
- 书源列表用首字母渐变图标，比纯文字更易识别
