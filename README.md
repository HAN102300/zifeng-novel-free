## 🚀 快速开始

### 📋 **前置要求**

- 🐰 **Node.js 18+**
- 📦 **npm 或 yarn**
- ☕ **Java 17+** (用于 SpringBoot 后端)
- 🗄️ **MySQL 8.0+**
- 📍 **Redis 7.0+**
- 💻 **现代浏览器**

### 🛠️ **本地开发**

#### 方案一：仅前端 + Node.js 后端

```bash
# 1️⃣ 克隆项目
git clone https://github.com/HAN102300/zifeng-novel-free.git
cd zifeng-novel-free

# 2️⃣ 安装前端依赖
npm install

# 3️⃣ 启动 Node.js 后端服务
cd server
npm install
npm start &

# 4️⃣ 启动前端开发服务器
cd ..
npm run dev

# 🎉 访问 http://localhost:5173
```

#### 方案二：完整 SpringBoot 后端

```bash
# 1️⃣ 启动 MySQL 和 Redis
# 可以使用 Docker Compose
cd backend
docker-compose up -d

# 2️⃣ 构建并启动 SpringBoot 后端
./mvnw clean package
java -jar target/zifeng-novel-0.0.1-SNAPSHOT.jar &

# 3️⃣ 启动前端
cd ..
npm run dev

# 🎉 访问 http://localhost:5173
```

### 🏗️ **生产部署**

#### 方案一：单服务器部署（推荐）

```bash
# 构建前端
npm run build

# 启动 Node.js 后端（包含静态文件服务）
cd server
npm install
npm start

# 访问 http://your-domain:3001
```

#### 方案二：完整 SpringBoot 部署

```bash
# 构建前端
npm run build

# 构建 SpringBoot 后端
cd backend
./mvnw clean package

# 启动服务
docker-compose up -d
java -jar target/zifeng-novel-0.0.1-SNAPSHOT.jar

# 访问 http://your-domain:8080
```

#### 方案三：Docker 全栈部署

```dockerfile
FROM node:18-alpine AS frontend
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
RUN npm run build

FROM openjdk:17-jdk-alpine AS backend
WORKDIR /app
COPY backend/pom.xml backend/
COPY backend/src backend/src
RUN cd backend && ./mvnw clean package

FROM openjdk:17-jdk-alpine
WORKDIR /app
COPY --from=frontend /app/dist ./static
COPY --from=backend /app/backend/target/zifeng-novel-0.0.1-SNAPSHOT.jar ./
EXPOSE 8080
CMD ["java", "-jar", "zifeng-novel-0.0.1-SNAPSHOT.jar"]
```

```bash
docker build -t zifeng-novel-reader .
docker run -p 8080:8080 zifeng-novel-reader
```

## 📖 使用指南

### 🎯 **基础使用**

1. **📖 开始阅读**
   - 访问首页浏览推荐书籍
   - 使用搜索功能查找目标小说
   - 点击书籍封面进入详情页
   - 点击「开始阅读」进入阅读界面

2. **⚙️ 配置书源**
   - 进入「书源管理」页面
   - 导入预设书源或自定义添加
   - 启用/禁用特定书源
   - 测试书源可用性

3. **👤 用户功能**
   - 注册/登录账号
   - 开启「记住我」功能（3天免登录）
   - 同步阅读进度到云端

4. **📱 个性化设置**
   - 调整字体大小和行间距
   - 切换明亮/暗黑主题
   - 设置阅读背景颜色
   - 配置书架显示方式

### 🔧 **高级功能**

#### 书源规则说明

```javascript
// 📋 JSONPath 规则示例
{
  "bookList": "$.data[*]",           // 获取所有书籍
  "name": "$.novelName",             // 书名
  "author": "$.authorName",          // 作者
  "coverUrl": "$.cover",             // 封面
  "intro": "$.summary"               // 简介
}

// 🎨 CSS选择器规则示例
{
  "bookList": ".book-item",          // 书籍列表
  "name": ".title@text",            // 书名
  "author": ".author@text",         // 作者
  "coverUrl": ".cover@src"           // 封面
}

// ⚡ JavaScript规则示例
{
  "content": "<js>\n// 自定义JS处理\nlet content = result;\n// 解密或处理内容\nreturn content;\n</js>"
}

// 🔒 加密内容解密示例
{
  "chapterUrl": "$.path@js:java.aesBase64DecodeToString(result,\"key\",\"AES/CBC/PKCS5Padding\",\"iv\")"
}
```

### 🔐 **安全特性**

- **JWT 认证** - 无状态登录，支持记住我功能
- **CORS 安全** - 配置允许的源，防止跨站请求
- **API 限流** - 防止暴力请求
- **JS 沙箱** - 安全执行自定义 JS 规则
- **环境变量** - 敏感配置外部化
- **参数验证** - 防止注入攻击
