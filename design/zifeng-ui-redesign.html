🧭 交互特性

  • 玻璃磨砂表面：backdrop-filter: blur(18px) saturate(160%)，半透明浮于内容之上，背景流光透出
  • 滚动响应收缩：页面滚动时导航栏 padding 减小、阴影加深、背景不透明度提升（见上方实际效果，向下滚动试试）
  • 磁吸下划线指示器：菜单项切换时，渐变下划线以弹簧曲线滑动到目标项，带紫色光晕
  • Logo 流光扫过：紫枫 Logo 标记每 4 秒有一道高光扫过，强化品牌记忆
  • 全局搜索：圆角胶囊搜索框，聚焦时紫色光晕环 + 背景提亮
  • 工具区：主题色切换 / 暗色模式 / 用户头像（hover 微旋转放大）/ 移动端汉堡菜单
  • 响应式：≤880px 隐藏菜单与搜索，显示汉堡按钮

🎬 动画令牌速查

  • --zf-ease-out cubic-bezier(.16,1,.3,1) — 通用出场
  • --zf-ease-spring cubic-bezier(.34,1.56,.64,1) — 弹簧反馈（下划线、按钮）
  • --zf-dur-fast .18s / --zf-dur-normal .32s / --zf-dur-slow .6s
  • 页面级：meshShift(背景网格 18s) · float1/2/3(光球 16-22s) · cardEnter(错峰入场) · reveal(滚动揭示)
  • 组件级：logoShine · shimmer · gradFlow · progressShine · livePulse · pulseGold · cardEnter
  • 交互级：hover 上浮缩放 · 磁吸下划线滑动 · 搜索聚焦光晕环 · 按钮光泽扫过
开发者对接
令牌 → CSS 变量 → Ant Design ConfigProvider 主题映射

🔧 实现路径

  • 令牌层：将 :root 中的 --zf-* 变量迁入 zifeng-web/src/index.css，替换现有散落硬编码色值
  • 主题映射：Ant Design ConfigProvider 的 theme.token 映射 colorPrimary=#8B5CF6、borderRadius=12、fontSize=14，与令牌对齐
  • 玻璃工具：复用已有 utils/glassStyle.js，新增 glassNavbar() 收缩态、glassNavIndicator() 磁吸下划线
  • 导航栏：替换 App.jsx Header，菜单项改用按钮 + framer-motion layoutId 实现磁吸下划线，滚动监听加 .scrolled
  • 小说卡：提炼 NovelCard 为独立组件，统一封面比例 200×280、错峰 animation-delay、Top3 金银铜徽章
  • 动画：keyframes 集中到 animations.css，Framer Motion 处理布局动画（下划线、列表过渡），CSS 处理装饰动画（光球、扫光）
  • 响应式：保留现有 6/5/4/3/2 列断点，导航栏 ≤880px 切汉堡菜单
  • 无障碍：所有交互元素 :focus-visible 紫色描边、按钮 ≥38px 触达区、动画遵守 prefers-reduced-motion

♿ 无障碍 & 性能

  • 文本对比度：正文 #F8FAFC / 背景 #0B0814 ≈ 17:1，远超 WCAG AA 4.5:1
  • 品牌紫 #8B5CF6 在暗底上用于大标题与图标，次要文本用 #CBD5E1 保证可读
  • 玻璃模糊仅在导航栏/Hero/卡片使用，避免大面积 backdrop-filter 影响滚动性能
  • 装饰光球 pointer-events:none 且 will-change:transform，GPU 加速
  • 骨架屏优先于 Spinner，减少布局抖动
