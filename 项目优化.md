## 2. React Bits 介绍

### 2.1 基本信息

| 项目 | 说明 |
|------|------|
| 仓库 | [github.com/DavidHDev/react-bits](https://github.com/DavidHDev/react-bits) |
| Star | 39.7k |
| 许可证 | MIT + Commons Clause |
| 组件总量 | **110+**（持续增长） |

### 2.2 核心特点

- **💬 Text Animations** — 文字入场、光泽、解密等动效
- **🌀 Animations** — 鼠标跟踪磁吸、光标特效等通用动画
- **🧩 Components** — UI 交互动效组件
- **🖼️ Backgrounds** — 粒子/波浪等动态背景
- **每个组件 4 种变体** — JS-CSS / JS-TW / TS-CSS / TS-TW
- **无外部依赖** — 仅依赖 framer-motion 或核心 React 能力
- **复制即用** — 复制源码到项目即可使用

### 2.3 本方案选用的 6 个组件

| 组件 | 类型 | 依赖 | 核心能力 |
|------|------|------|----------|
| `BlurText` | TextAnimate | framer-motion | 文字逐词/逐字模糊淡入，IntersectionObserver 触发 |
| `CountUp` | TextAnimate | framer-motion | 数字从 from 滚动到 to，支持 spring 物理动画 |
| `ShinyText` | TextAnimate | framer-motion | 光泽扫过文字，支持 delay/方向/悬停暂停 |
| `DecryptedText` | TextAnimate | framer-motion | hover/click/view 触发解密效果 |
| `TrueFocus` | TextAnimate | framer-motion | 焦点框在文字间自动轮播/悬停跟随 |
| `Magnet` | Animation | 无 | 鼠标磁吸跟随效果 |

## 3. 安装与集成

### 3.1 安装方式

```bash
# 安装 jsrepo（已安装）
npm install -D jsrepo

# 配置注册表（你已配置）
npx jsrepo init react-bits https://reactbits.dev/r

# 安装具体组件
npx jsrepo add react-bits/BlurText-JS-CSS
```

### 3.2 当前集成状态

组件已复制到项目 `zifeng-web/src/components/react-bits/` 目录下，可直接导入：

```jsx
import { BlurText, CountUp, ShinyText } from '../components/react-bits'

// 使用示例
<BlurText
  text="紫枫免费小说"
  delay={150}
  animateBy="words"
  direction="top"
/>
```

对于 `zifeng-admin`（管理后台），可直接从 `zifeng-web` 的组件目录中复制需要的组件。

## 4. 现有动画架构分析

### 4.1 当前使用的动画技术

| 技术 | 用途 |
|------|------|
| **framer-motion** | 页面路由入场（`AnimatePresence`）、hover 微交互、卡片入场 |
| **animejs** | 后台工具函数（`fadeInUp`, `staggerFadeIn`, `cardHover` 等）zifeng-booksource 页面使用 |
| **CSS transition** | 侧边栏折叠展开、图片悬停放大 |

### 4.2 当前不足

| 问题 | 说明 |
|------|------|
| 路由动画同质化 | 所有路由使用完全相同的 `y 20 ↔ -20` 动画 |
| 无文字入场动效 | 标题、书名等文字直接渲染，无任何入场动画 |
| 统计数字无滚动 | dashboard 统计卡片虽有 stagger 入场，数字直接更新无动画 |
| 非交互性 | 页面缺乏鼠标跟随等新奇交互体验 |
| 品牌感弱 | 首页无 Hero 区域，品牌名称无视觉效果 |
| 空状态/加载态单调 | Spin + 空白表格，无趣味性 |

## 8. 附录：已安装组件 API 速查

### 8.1 BlurText

```jsx
<BlurText
  text="要展示的文字"           // string
  delay={150}                   // number - 每项延迟 ms
  animateBy="words"             // "words" | "characters"
  direction="top"               // "top" | "bottom"
  threshold={0.1}               // number - IntersectionObserver
  rootMargin="0px"              // string
  animationFrom={{ filter: 'blur(10px)', opacity: 0, y: -50 }}
  animationTo={[{ filter: 'blur(5px)', opacity: 0.5, y: 5 }, { filter: 'blur(0px)', opacity: 1, y: 0 }]}
  easing={t => t}               // function - 缓动函数
  onAnimationComplete={() => {}} // function
  stepDuration={0.35}           // number
/>
```

### 8.2 CountUp

```jsx
<CountUp
  to={1000}                     // number - 目标值
  from={0}                      // number - 起始值
  direction="up"                // "up" | "down"
  delay={0}                     // number - 延迟（秒）
  duration={2}                  // number - 动画时长（秒）
  startWhen={true}              // boolean
  separator=""                  // string - 千分位分隔符
  onStart={() => {}}            // function
  onEnd={() => {}}              // function
/>
```

### 8.3 ShinyText

```jsx
<ShinyText
  text="文字内容"               // string
  disabled={false}              // boolean
  speed={2}                     // number - 动画速度
  color="#b5b5b5"               // string - 基础色
  shineColor="#ffffff"          // string - 光泽色
  spread={120}                  // number - 渐变角度
  yoyo={false}                  // boolean - 来回扫
  pauseOnHover={false}          // boolean
  direction="left"              // "left" | "right"
  delay={0}                     // number - 延迟（秒）
/>
```

### 8.4 DecryptedText

```jsx
<DecryptedText
  text="文字内容"               // string
  speed={50}                    // number
  maxIterations={10}            // number
  sequential={false}            // boolean - 顺序/随机解密
  revealDirection="start"       // "start" | "end" | "center"
  animateOn="hover"             // "hover" | "click" | "view" | "inViewHover"
  clickMode="once"              // "once" | "toggle"
  characters="ABC..."           // string - 加密字符集
/>
```

### 8.5 TrueFocus

```jsx
<TrueFocus
  sentence="焦点文字"           // string - 以分隔符拆分的句子
  separator=" "                 // string
  manualMode={false}            // boolean - 手动/自动
  blurAmount={5}                // number - 非选中词模糊度
  borderColor="green"           // string - 边框色
  glowColor="rgba(0,255,0,0.6)" // string - 发光色
  animationDuration={0.5}       // number - 动画秒数
  pauseBetweenAnimations={1}    // number - 间隔秒数
/>
```

### 8.6 Magnet

```jsx
<Magnet
  padding={100}                 // number - 感应范围
  disabled={false}              // boolean
  magnetStrength={2}            // number - 磁力强度
  activeTransition="transform 0.3s ease-out"
  inactiveTransition="transform 0.5s ease-in-out"
>
  <YourComponent />             // children
</Magnet>
```
